Questions fréquentes (FAQ)

Parcourez nos questions les plus courantes ou recherchez une réponse spécifique.

Qu'est-ce qu'AslazMarket ?

AslazMarket est une marketplace B2B et B2C internationale qui connecte directement les acheteurs avec des vendeurs vérifiés. Notre plateforme permet aux vendeurs de publier leurs produits et aux acheteurs de les contacter directement via WhatsApp ou appel téléphonique pour finaliser leurs achats.

Comment fonctionne AslazMarket ?

Le fonctionnement est simple :

  1. Les vendeurs créent un compte et publient leurs produits avec photos et descriptions
  2. Les acheteurs parcourent les catégories et trouvent des produits qui les intéressent
  3. Lorsqu'un acheteur clique sur "Acheter", il est redirigé vers le WhatsApp ou le numéro de téléphone du vendeur
  4. L'acheteur et le vendeur finalisent la transaction directement
AslazMarket est-il gratuit ?

Oui, l'utilisation d'AslazMarket est totalement gratuite pour les acheteurs. Pour les vendeurs, la création de compte et la publication des produits sont gratuites. Nous appliquons une petite commission uniquement sur les ventes effectuées via la plateforme.

Comment contacter un vendeur ?

Pour contacter un vendeur :

  1. Trouvez le produit qui vous intéresse
  2. Cliquez sur le bouton "Contacter le vendeur" ou "Acheter maintenant"
  3. Vous serez automatiquement redirigé vers WhatsApp du vendeur ou pourrez l'appeler directement
  4. Discutez avec le vendeur pour finaliser votre commande
Comment savoir si un vendeur est fiable ?

Plusieurs indicateurs de fiabilité :

  • Badge "Vérifié" : Le vendeur a fourni ses documents d'identité
  • Évaluations : Consultez les avis des autres acheteurs
  • Ancienneté : Les vendeurs présents depuis longtemps sont généralement plus fiables
  • Réactivité : Un vendeur qui répond rapidement est un bon signe
Comment devenir vendeur sur AslazMarket ?

Pour devenir vendeur :

  1. Créez un compte sur AslazMarket
  2. Cliquez sur "Devenir vendeur" dans votre espace compte
  3. Complétez le formulaire d'inscription vendeur
  4. Fournissez les documents requis (pièce d'identité, etc.)
  5. Une fois votre compte validé, vous pouvez commencer à publier vos produits
Quels sont les frais pour les vendeurs ?

Les frais pour les vendeurs sont :

  • Inscription : Gratuite
  • Publication de produits : Gratuite
  • Commission : 5% sur chaque vente effectuée via la plateforme
  • Frais de transaction : Dépendent de votre méthode de paiement
